# ü§ñ DSA Mentor AI

**DSA Mentor AI** is a command-line Python application that acts as your personal tutor for Data Structures and Algorithms (DSA). It uses Google's Gemini 1.5 Flash model to help beginners learn key DSA topics in a simple, interactive, and conversational way.

The assistant:

- Teaches you core DSA concepts (like Arrays, Trees, Graphs, etc.)
- Walks you through Python coding problems line by line
- Provides alternate solutions
- Links relevant LeetCode practice questions
- Suggests quality study material (blogs, videos, docs)
- Lets you ask follow-up questions in a chat-style format
